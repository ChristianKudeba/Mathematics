# Calkin–Wilf tree

## Definition

Binary tree of pairs of positive integers, generated by:
- Root: $(1,1)$
- Left child: $(a, b) \mapsto (a, a+b)$
- Right child: $(a, b) \mapsto (a+b, b)$

```
                    (1,1)
              /             \
           (1,2)            (2,1)
          /     \          /     \
       (1,3)  (3,2)    (2,3)   (3,1)
       ...
```

## Key facts (Calkin–Wilf 2000)

1. **Enumerates ℚ_{>0}**: each positive rational $a/b$ in lowest terms appears exactly once when reading the tree row-by-row left-to-right.
2. **First-component sequence is Stern's diatomic** $a(n)$ (OEIS A002487):
$$a(2n) = a(n),\qquad a(2n+1) = a(n) + a(n+1)$$
with $a(1) = 0$, $a(2) = 1$. Pairs are $(a(n), a(n+1))$ row-by-row.
3. **Stern's sequence is [[15-k-regular-sequences|2-regular]]**.

## Connection to Shakov's $\Phi_0$ tree

Both are binary trees on pairs of natural numbers, generated by linear-fractional-style maps. Comparing:

| | $\bar L, \bar R$ on Calkin–Wilf | $\bar S, \bar T_{\phi_0}$ on $\mathcal{D}_{\phi_0}$ |
|---|---|---|
| left | $(a, b) \to (a, a+b)$ | $(m, n) \to (m, m+n)$ |
| right | $(a, b) \to (a+b, b)$ | $(m, n) \to ?$ (depends on $f$) |
| root | $(1,1)$ | $(1,0)$ |
| Aim | enumerate ℚ⁺ | enumerate $\mathcal{D}_{\phi_0}$ |

**Note the perfect parallel between $\bar L$ and $\bar S$**! The "left-child" map is identical (up to root). The crucial difference is the right child: Calkin–Wilf uses the same shape, but $\bar T_{\phi_0}$ is a *complemented* version that depends on the polynomial.

## Same row-ratio mean

Theorem (Han–Masuda–Singh–Thiel 2019): the limiting average of the row ratios $a/b$ across the Calkin–Wilf tree converges to $3/2$. The paper proves (Cor `phi0average`) the *same* limit for the ratio mean over $\Phi_0$ rows.

This is **not** a coincidence: both trees have a similar self-similar structure under doubling, and the $3/2$ is a Stern-like average. It hints at a deeper isomorphism between Calkin–Wilf and the $\Phi_0$ trees in some asymptotic sense.

## Question: is there a "Stern's-sequence-style" closed form?

Stern's $a(n)$ has the beautiful closed form:
$$a(n) = \#\{\text{representations of } n-1\ \text{as a hyperbinary sum}\}$$

(hyperbinary = sum of distinct powers of 2 where each power can be used 0, 1, or 2 times)

**Open question**: does $\mathcal{S}(k)$ have a similarly elegant combinatorial description? The recursion structure suggests it may, but the paper doesn't settle this.

If $\mathcal{S}(k)$ admits such a description, it would give a *direct* combinatorial route to the [[14-S-sequence|fiber-count]] question for Landau's 4th.

## Structural lesson

Calkin–Wilf is the "**non-divisor**" version of $\Phi_0$ — the version where every pair is allowed (every coprime pair $(a, b)$ appears). $\Phi_0$ is the "**divisor-restricted**" version: only pairs $(m, n)$ with $m \mid n^2+1$. The restriction is severe: among the $\sim N^2$ coprime pairs with $a, b \le N$, only $\sim N \cdot \log N$ are valid divisor pairs.

The fact that $\Phi_0$ enumerates the *restricted* set bijectively via the *same* free-monoid skeleton as Calkin–Wilf is the **content of Diophantus' identity**. It is the "free monoid SL₂(ℕ₀)" that magically respects the divisor restriction for the four enumerable polynomials.
