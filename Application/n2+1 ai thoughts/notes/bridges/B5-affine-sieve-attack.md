# Affine-sieve attack on Landau's 4th

## Setup

The [[../concepts/13-prime-characterization|prime characterization]] for $\phi_0$:
$$n^2 + 1\ \text{is prime} \iff n \notin \chi(SL_2(\mathbb{N}))$$
where $\chi(A) = ac + bd$ and $SL_2(\mathbb{N})$ is the **interior semigroup**.

This is structurally the same as: showing that a thin orbit FAILS to cover certain integers infinitely often.

## Why this looks like an affine-sieve problem

The [[../research/R5-thin-groups-affine-sieve|affine sieve]] of Bourgain–Gamburd–Sarnak studies orbits $\Gamma . v_0$ where $\Gamma \subset GL_n(\mathbb{Z})$ is a thin (Zariski-dense, infinite-index) subgroup, with goal:

> Detect almost-primes among polynomial values $f(\gamma . v_0)$ for $\gamma \in \Gamma$.

In our setting:
- $\Gamma = SL_2(\mathbb{N})$ (semigroup, not group — modification needed)
- $v_0 = $ no specific vector; instead the polynomial is $\chi(A) = ac+bd$ on the matrix itself
- The "thin orbit" is $\chi(SL_2(\mathbb{N})) \subset \mathbb{N}$

We want: "almost primes" in the **complement** of $\chi(SL_2(\mathbb{N}))$.

## Critical obstacle 1: semigroup, not group

The whole BGS machinery uses **superstrong approximation**: the Cayley graphs of $\Gamma$ in $SL_2(\mathbb{Z}/p)$ are expanders, providing a spectral gap.

For semigroups, this is **subtle**. Bourgain–Gamburd's expansion proof uses the trace inequality and Helfgott-style growth, which require both an element and its inverse. Without inverses, the standard expansion isn't immediately available.

**Workaround**: pass to the group generated. $SL_2(\mathbb{Z})$ is generated by $S$ and $T$ as a group (with relations modulo center). Inside SL₂(ℤ), the orbit data of SL₂(ℕ) can be recovered from SL₂(ℤ) data by a positivity selection — the cross-term $\chi(A) = ac + bd$ is positive *exactly* on $SL_2(\mathbb{N}_0)$. So we can work in SL₂(ℤ) and project.

Actually a key obstacle: SL₂(ℤ) is NOT a thin group — it's the full lattice. Standard BGS thin-group expansion is not directly applicable; for SL₂(ℤ) we have the much sharper Selberg/Iwaniec spectral theory of automorphic forms instead.

## The connection to Iwaniec's half-dimensional sieve

Iwaniec's 1978 result $\pi_{\phi_0}^{P_2}(N) \gg N^{1/2}/\log N$ uses the half-dimensional sieve, which is parameterized by:
$$\sum_{\substack{p \le z \\ \chi(p) = 1}} \frac{1}{p} \sim \frac{1}{2} \log\log z$$
where $\chi$ here is the Kronecker symbol $\left(\frac{-1}{p}\right)$. The "1/2" is the dimension.

In Shakov's framework, this dimension should appear as the **spectral exponent** of a transfer operator. Specifically:
$$\chi(SL_2(\mathbb{N})) \cap [1, N] \sim ? \cdot N^?$$

If the orbit count grows like $N \cdot (\log N)^{\alpha}$ for some $\alpha$, then the *complement* (the Landau-prime candidates) has predicted density $C N / (\log N)^{1+\alpha} \cdot (\text{something})$. For this to match Hardy–Littlewood ($\sqrt N / \log N$), we'd need $\alpha = -1/2$ — exactly the half-dim sieve dimension.

So the half-dim sieve is implicitly computing the **second moment** of fiber size $|\mathcal{S}^{-1}(n)|$. This is encouraging — Iwaniec's $P_2$ result and the SL₂(ℕ₀) framework should be talking about the same spectral object.

## Critical obstacle 2: parity problem

Even with full spectral data, the [[../research/R2-parity-problem|parity problem]] obstructs going from $P_2$ to $P_1$. **The Shakov framework does not automatically circumvent this.**

The parity problem says: any "linear sieve" can't distinguish *odd-prime-factor-count* from *even-prime-factor-count*. So sums like $\sum_{n \le N} \lambda(n^2+1)$ (with $\lambda$ the Liouville function) are invisible to sieves. A sieve sees both prime $n^2+1$ and $n^2+1 = pq$ with the same weight.

To break parity, you need a **bilinear estimate** — a Type II sum showing genuine cancellation. Friedlander–Iwaniec's breakthrough for $a^2 + b^4$ used such an estimate via the $b$-variable. For $n^2+1$, there is no second variable to "open up."

**The Shakov hope**: the SL₂(ℕ₀) parametrization $A \mapsto \chi(A)$ EXPOSES bilinear structure. Specifically, $\chi(A) = ac + bd$ is **bilinear** in $(a,b)$ and $(c,d)$ as the row-vectors of $A$. The det constraint $ad - bc = 1$ couples them, but the bilinearity is real.

This is the most promising single observation in the Shakov framework. **Bilinear sums in SL₂(ℕ₀) words might break parity for $n^2+1$.** This needs careful work.

## Critical obstacle 3: density 1/√N

$\phi_0$ is a degree-2 polynomial, so primes appear in a thin set of density $1/\sqrt N$. Most affine-sieve breakthroughs are for *positive-density* sets in their natural parameter space.

**Workaround**: don't sieve directly on $n$, sieve on $A \in SL_2(\mathbb{N})$. The matrix space has higher dimension; matrices of word-length $\le k$ number $2^k$, while the cross-term values they produce span $[k, F_{2k}]$. So the *matrix-side density* is essentially $\sqrt{N \log N}$ matrices per integer $\le N$ — much fatter.

Run a sieve on matrices, then project. This is exactly the Bourgain–Kontorovich strategy for Zaremba.

## Concrete program (sketch)

1. **Define a transfer operator** $\mathcal{L}$ on functions $f : \mathcal{D}_{\phi_0} \to \mathbb{C}$ that encodes the action of $\bar S$ and $\bar T_{\phi_0}$.
2. **Compute its spectrum**. By analogy with the [[../concepts/18-row-recursions|row recursions]], the dominant eigenvalue should be $(5+\sqrt{17})/2$.
3. **Pass to congruence quotients**: $\mathcal{L}_q$ on $SL_2(\mathbb{N}_0)/q$. Show spectral gap (this is the hard part — needs adaptation of expansion proofs to semigroups, perhaps through SL₂(ℤ)).
4. **Run a sieve** to count $\#\{A \in SL_2(\mathbb{N}) : \chi(A) = p, p\ \text{prime}\}$. This counts matrices whose cross-term is itself a prime.
5. **Cross-term complementary count**: bound $\#\{n \le N : n \notin \chi(SL_2(\mathbb{N}))\}$ from below using the sieve in step 4 + duality.

Steps 1-3 are likely tractable with current technology. Step 4-5 are where the parity problem hits, and where the bilinear structure of $\chi$ might save us.

## Why this might actually work

Three reasons for cautious optimism:

1. **Bilinear cross-term**. As noted, $\chi(A) = (a,b) \cdot (c,d)^T$ is genuinely bilinear, providing built-in Type-II structure.

2. **Free monoid**. SL₂(ℕ₀) is a *free* monoid — every matrix has a unique S/T-word. This is much sharper than typical thin-group settings (where word problems are nontrivial). All sums can be reduced to **sums over words**, which can be analyzed via [[../research/R7-k-regular-asymptotics|transfer operator]] techniques.

3. **Quadruple coupling**. Each matrix produces divisor information for **four** polynomials simultaneously ([[../concepts/08-the-four-polynomials|the highway]]). Joint sieving over all four might break the parity barrier for at least one.

This is the program. See [[B6-open-questions]] for sub-problems to actually start working on.
